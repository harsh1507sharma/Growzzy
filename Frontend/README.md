# 🚀 Growzzy: Next-Generation EdTech Platform

**Growzzy** is a premium, high-fidelity EdTech platform built to transform the way education is managed, monitored, and delivered. With a sleek, dark-themed, and control-room-inspired interface, Growzzy bridges the gap between active learning, educator guidance, and parental oversight.

---

## ✨ Features

### 🛡️ Guardian Dashboard
- **Parental Oversight Console:** A high-tech monitoring suite that keeps parents informed of their child's intellectual journey.
- **Knowledge Monitoring:** A robust bento-grid tracking system that visualizes Learning Hours, Mastery Levels, and Path Velocity.
- **AI Insights:** Synthesized, real-time feedback detailing the child's academic affinities (e.g., spatial logic, micro-biology engagement) and recommending modular curriculum additions.
- **Active Mentors:** Direct access to tracking the specialized mentors currently supervising coursework.

### 👩‍🏫 Educator Workspace
- **Command Console:** A fluid layout tailored specifically for educators to track performance metrics and deliver personalized guidance effectively.
- **Dynamic Curriculum Builder:** Seamlessly manage advanced modules like "Advanced Quantum Physics" or "Lexicon Mastery".

### 🎨 Premium UI/UX
- Responsive, modular **Bento-style layouts**.
- Vibrant, futuristic typography and visual accents (glassmorphism details, glowing borders, tailored dark modes).
- Developed with seamless **Tailwind CSS** implementations alongside **Lucide React** icons.

---

## 🛠️ Technology Stack

- **Frontend Framework:** React (Vite)
- **Styling:** Tailwind CSS (Custom Color Palettes & Utilities)
- **Icons:** Lucide-React
- **Architecture:** Modular Functional Components separated into global hooks, structural `pages`, and granular `components`.

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/harsh1507sharma/Growzzy.git
```

### 2. Install Dependencies
Ensure you have Node.js installed, then navigate into the frontend directory:
```bash
cd Growzzy/Frontend
npm install
```

### 3. Run Development Server
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:5173` (or the port provided in the terminal) to view the application.
